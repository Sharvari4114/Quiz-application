# Quiz-application
Spark Quiz is a Java-based desktop quiz application developed using Java Swing. It provides an interactive and user-friendly platform for testing programming knowledge in Java, Python, and C++. The application collects candidate details before the quiz, randomly selects questions based on the chosen subject, evaluates answers, calculates the final score, and stores quiz results with the candidate's information and timestamp in a local file.

Key Features
📝 Google Forms–style registration page for candidate details
👤 Collects candidate name and email before starting the quiz
📚 Subject selection (Java, Python, and C++)
🎲 Random selection of 10 questions from a larger question bank
⏱️ Countdown timer with progress bar
✅ Multiple-choice questions with instant navigation
📊 Automatic score calculation
💾 Saves quiz results (Name, Email, Score, and Timestamp) to a local text file
🎨 Modern and responsive Java Swing user interface
🔀 Randomized question order for every quiz attempt


Technologies Used
Java
Java Swing
AWT
Object-Oriented Programming (OOP)
